# Assignment Management System

This is a simple web application created as a student project.

## Features
- Teacher can post assignments
- Students can upload submissions
- Teacher can evaluate and give marks
- Mentor can view student performance

## Tech Stack
- React
- Node.js
- Express
- PostgreSQL

## Run the project

npm install
npm run dev

Then open http://localhost:5173
